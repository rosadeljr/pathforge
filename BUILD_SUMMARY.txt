HOW TO START:

1. npm install
2. cp .env.example .env.local
3. Get Supabase URL/keys and paste in .env.local
4. In Supabase: Run migrations and seed SQL
5. npm run dev
6. Visit http://localhost:3000

TO DEPLOY:
1. git push to GitHub
2. Connect to https://vercel.com/new
3. Add env variables
4. Click Deploy
